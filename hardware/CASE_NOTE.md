# About the skiff case

The 3U/28HP skiff enclosure used in this build (`case/EURORACK_SKIFF_3U_28HP_41mm_height.stl`)
is a **remix of a third-party design**, resized/adapted for this project.

- **Original design:** [Eurorack Skiff 32 HP / 55mm with Decksaver](https://makerworld.com/models/2244685)
- **Original author:** [Jaschasuess](https://makerworld.com/en/@jaschasuess) (MakerWorld)
- **Original license:** Creative Commons Attribution 4.0 ([CC BY 4.0](https://creativecommons.org/licenses/by/4.0/))
- **What was changed:** resized/adapted to 3U / 28HP / 41 mm height for this
  build's panel and internal clearance needs.

CC BY 4.0 permits sharing modified versions, including commercially, as long
as the original is credited — which this file does. The remixed STL is
included in `hardware/case/`.

If you print this, consider also crediting the original designer wherever
you share photos or files of your own build.
